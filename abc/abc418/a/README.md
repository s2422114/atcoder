# ABC418 A - You're a teapot

- URL: https://atcoder.jp/contests/abc418/tasks/abc418_a
- 結果: AC（制約の見落としで最初落ちる→長さチェック追加で修正）
- 解法: S の末尾3文字が t,e,a か判定。ただし長さ3未満を先に弾く
    if (S.size() >= 3 && 末尾3文字が tea) Yes
- 詰まった点と学び:
  - 制約 1<=N<=20。N=1,2 のとき S.size()-3 が範囲外（size_tなので巨大な数に）
    → RE。長さ3未満のケースを弾く必要があった
  - S.size() >= 3 && を「一番前」に置く。&& は左がfalseなら右を評価しない
    ので、危ない .at() の前に安全確認を置くと範囲外を踏まない
  - 別解: S.substr(S.size()-3) == "tea"（substrで末尾3文字を取り出す）
  - 教訓: 制約の下限(一番小さいケース)を必ず確認する
