# ABC420 A

- URL: https://atcoder.jp/contests/abc420/tasks/abc420_a
- 結果: AC（自力）
- 解法(提出): X+Y<=12 ならそのまま、超えたら 12 を1回引く（今の制約なら十分）
- より一般的な解法: ans = (X+Y) % 12; if(ans==0) ans=12;
  - % を使えば何周していても一発で範囲に収まる（12を何回でも引くのと同じ）
- 学び:
  - 1〜12 で回る周期は % 12 で扱えるが、% は 0 始まり(0〜11)で結果を返す
    → ちょうど12の倍数のとき 0 になる。「0月」は無いので 12 に読み替える
  - 「1始まりの周期を % で扱うときは、0なら最大値に直す」定番処理
  - 【気づき】提出した if 版は今の制約では正解だが、Y が大きいと引き算1回では
    足りない。制約が変わったら % が必要 → 解法の適用範囲を意識できた
