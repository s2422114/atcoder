# ABC421 A - Misdelivery

- URL: https://atcoder.jp/contests/abc421/tasks/abc421_a
- 結果: AC（読み違いで1度WA→修正）
- 解法: S を 1-indexed で受け取り、S[X] == Y なら Yes。ループ不要
- 詰まった点と学び:
  - 「X号室の住人がYか」を「どこかにYがいるか」と読み違えて全探索してしまい、
    出力例4（別部屋に同名がいるケース）で不一致
  - 正しくは S.at(X) を直接見るだけ。for も check も不要
  - 1-indexed で作ったので S.at(X) がそのまま X 号室（-1調整が不要）
  - 教訓: 与えられた入力(X)を使っていない = 読み違いのサイン
    → ひっかけ対策として「各入力を何に使うか」を確認する
