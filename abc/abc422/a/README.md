# ABC422 A

- URL: https://atcoder.jp/contests/abc422/tasks/abc422_a
- 結果: AC（文字↔数の変換と出力を相談）
- 解法: S="i-j" を受け取り、i=S[0]-'0', j=S[2]-'0' で数に変換。
        j<8 なら (i, j+1)、j=8 なら (i+1, 1) を出力
- 学び:
  - 文字→数: S[0]-'0'（'4'-'0'=4。文字コードの差を使う定番）
  - 数→出力: cout << i << "-" << (j+1) で直接流す（文字列を組み立てない）
  - 連結の罠: i + '-' + j は「数の足し算」になってダメ。
    文字列で組むなら to_string(i)+"-"+to_string(j)
  - 制約「8-8は来ない」→ j=8 のとき i<8 が保証され、else だけで済む
  - 使わない変数の出力(cout << next_stage)は余計な空行の元。消す
