# ABC423 A - Scary Fee

- URL: https://atcoder.jp/contests/abc423/tasks/abc423_a
- 結果: AC（方針を相談）
- 解法: 1000円ブロック1個のコスト = 1000 + C（手数料は1000円あたりC円）
        買えるブロック数 = X / (1000 + C)（切り捨てが味方）
        答え = ブロック数 * 1000
- 学び:
  - 「1セット/1ブロックいくらか → 何個買えるか」の割り算パターン
    ※ ABC434(風船), ABC428(走る時間)と同じ型。久しぶりで方針が出なかった
  - 手数料は「1回」でなく「1000円あたり」→ まとめても得しない（引っかけ）
  - 答えはブロック数でなく「×1000」して円に直す（単位変換を忘れない）
  - 切り捨てが味方になるパターン（余りでは1ブロック買えない）
