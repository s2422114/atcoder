# ABC434 A - Balloon Trip

- URL: https://atcoder.jp/contests/abc434/tasks/abc434_a
- 結果: AC（自力）
- 解法: 飛ぶ条件 W*1000 < n*B。これを満たす最小 n = W*1000 / B + 1
- 罠: 単位（W は kg → *1000 で g に換算）、「未満(<)」であること
- 学び【割り算のイディオム】:
  - 切り上げ（天井除算）: (A + B - 1) / B
      → A ≤ n*B（以下で成立）の最小 n がほしいとき
  - 超える最小: A / B + 1
      → A < n*B（未満で成立）の最小 n がほしいとき（今回）
  - 「以下」か「未満」かで +1 の要否が変わる。読み間違えると1ズレてWA
  - 式が不安ならループで愚直に（条件を満たすまで n++）
  - 小数の ceil は誤差リスク。個数問題は整数イディオムを使う
