# ABC435 A - Triangular Number

- URL: https://atcoder.jp/contests/abc435/tasks/abc435_a
- 結果: AC（自力）
- 解法: 三角数の公式。偶奇で場合分けして整数除算の切り捨てを回避
- 改善: N*(N+1) は連続2整数の積で必ず偶数 → 場合分け不要
    int ans = N * (N + 1) / 2;   // 掛けてから割れば切り捨てなし
- 別解: for で 1..N を素直に足す（N が小さいので十分。確実）
- 学び:
  - 三角数 = N*(N+1)/2。掛け算を先、割り算を後にすると切り捨てが起きない
  - 公式が不安なときは愚直 for に逃げてよい
  - N が大きいと N*(N+1) がオーバーフロー → long long を使う
