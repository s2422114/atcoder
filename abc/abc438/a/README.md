# ABC438 A

- URL: https://atcoder.jp/contests/abc438/tasks/abc438_a
- 結果: AC（自力）
- 解法: (D - F) % 7 で年末までの余り日数を求め、7 - remainder が翌年の日付
- 別解: 愚直ループ。day=F から day<=D の間 7 ずつ足し、day - D が答え
- 学び:
  - 「7日ごと」のような周期問題は「割ったあまり(%)」で位置を求められる
  - 問題文→数式への翻訳が難所。詰まったら具体例を数個手で並べて規則を探す
  - 式が見えないときは愚直ループに逃げてよい（制約が小さければ十分速い）
