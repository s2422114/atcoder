# ABC439 A

- URL: https://atcoder.jp/contests/abc439/tasks/abc439_a
- 結果: AC（自力）
- 解法: ループで2をN回かけて 2^N を作り、2^N - 2N を出力
- 学び:
  - 累乗は pow（小数を返す→誤差の罠）を避け、整数で計算する
  - 2^N はビットシフト 1 << N が最短（1を左にN回ずらす＝2をN回かける）
    例: 1<<3 は 0001→1000 で 8
  - 優先順位に注意: (1 << N) - 2*N のようにカッコで囲む
  - 大きなシフトは桁あふれ注意 → 1LL << N（long long）
