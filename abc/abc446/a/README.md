# ABC446 A

- URL: https://atcoder.jp/contests/abc446/tasks/abc446_a
- 結果: AC（自力）
- 解法: S.at(0) に小文字化した文字を代入し、先頭に "Of" を付けて出力
- 詰まった点:
  - 文字列の特定の位置は S.at(0) = 文字 で直接代入できることを忘れ、
    replace を使おうとしていた
  - 保存し忘れて古いバイナリが動いていた（'S' - 32 = '3' が出力されて混乱）
- 学び:
  - 文字列は文字の配列。S.at(i) = 'x' で直接書き換えられる
  - 大文字→小文字は +32（= 'a' - 'A'）。tolower() を使うと意図が明確
  - cout << "Of" << S << endl; で連結せずそのまま出力できる
