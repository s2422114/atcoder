# ABC447 A

- URL: https://atcoder.jp/contests/abc447/tasks/abc447_a
- 結果: AC（自力）
- 解法: N席に隣り合わず座れる最大人数を求め、M以下なら Yes
        偶数なら N/2、奇数なら N/2 + 1 と場合分け
- 別解: 切り上げ除算で場合分け不要
        if ((N + 1) / 2 >= M) → Yes
- 学び:
  - 切り上げ除算: a/b の切り上げは (a + b - 1) / b
    b=2 なら (N + 1) / 2 で偶数・奇数の両方をカバーできる
  - 整数除算は切り捨てられるので、切り上げたいときはこの形を使う
