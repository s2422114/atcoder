# ABC452 A

- URL: https://atcoder.jp/contests/abc452/tasks/abc452_a
- 結果: AC（自力）
- 解法: 五節句の月・日を配列に持ち、for で回して一致するか bool フラグで判定
- 学び:
  - 条件をベタ書きせず、データを配列に持たせてループで回すと、
    データとロジックが分離されて拡張しやすい
  - boolフラグの型: false で始めて、見つけたら true。出力はループの外で1回
- 解き終わってから気づいた別解:
  - 1月7日だけ特別で、残り（3,5,7,9月）は M == D という規則性がある
    if (M == 1 && D == 7) → Yes
    if (M == D && (M==3 || M==5 || M==7 || M==9)) → Yes
  - 規則性を見つけて条件をまとめる訓練は、C問題以降で効いてくる
