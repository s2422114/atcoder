# ABC456 A

- URL: https://atcoder.jp/contests/abc456/tasks/abc456_a
- 結果: AC（自力、時間内）
- 解法: サイコロ3個の全組み合わせ(6^3 = 216通り)を三重ループで全探索し、
        合計が X になるものがあるか bool フラグで判定
- 別解: 合計が取りうる範囲は 3〜18 で、その間の整数はすべて作れる
        → if (3 <= X && X <= 18) の範囲チェック1行で済む
- 学び:
  - 全探索でも計算量が間に合うなら正当な解法（216通りは一瞬）
  - 「まず全探索できるか計算量を見積もる → 間に合うなら全探索、
     間に合わないなら考察して探索を減らす」という判断順序が実戦的
