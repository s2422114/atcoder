# ABC461 B - Axes

- URL: https://atcoder.jp/contests/abc461/tasks/abc461_b
- 結果: AC（自力）
- 解法: 女神情報 b[i]=「斧iを持つ木こり」。木こり情報 a と突き合わせる。
        全 i で a[b[i]] == i なら矛盾なし → Yes、一つでも崩れたら No
- 学び【1-indexed】:
  - vector<int> a(N+1) はサイズ N+1（番号 0..N）。a[0] を使わず 1..N を使う
  - a[0] を放置してもエラーにならない（触らなければOK、メモリ1個ぶん余るだけ）
  - 問題が 1..N の番号を使うときは配列を1個大きく作り 1-indexed にすると、
    値をそのまま添字に使えて -1 が不要 → オフバイワンのバグを防げる
  - 範囲外アクセス（例 a[N+1]）は NG。.at() と _GLIBCXX_DEBUG が早期検知してくれる
