# ABC463 B - Seat Reservation

- URL: https://atcoder.jp/contests/abc463/tasks/abc463_b
- 結果: AC（★B問題 初の自力AC）
- 解法: X - 'A' で列をインデックス(0〜4)に変換し、各列車のその列に 'o' があるか判定
- 学び:
  - 文字→数値の変換: X - 'A' で 'A'〜'E' が 0〜4 になる
    （c - '0' で数字文字→整数、c - 'a' で小文字→0〜25 も同じ原理）
  - boolフラグ: false で始めて、見つけたら true にする（ABC466 A と同じ型）
- 改善点:
  - 内側の j ループは不要だった。idx で列を直接指定しているので、
    全マス走査ではなく「特定の列だけ縦に見る」で足りる
  - 二次元配列も不要。string で1行ずつ読んで、その場で S[idx] を判定すればよい
    （後で使わないデータは保存しない）
