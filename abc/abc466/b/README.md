# ABC466 B - Representative Balls

- URL: https://atcoder.jp/contests/abc466/tasks/abc466_b
- 復習: 要（初見で解けなかった。ans[c[i]] の入れ子が出てこなかった）
- 復習日: 未
- 結果: AC（初見では解けず、ヒントあり）
- 解法: 色番号を配列の添字にして、色ごとの最大サイズを保持
  - ans を -1 で初期化 → ボールが来なかった色は -1 のまま残る
  - ans[c[i]] = max(ans[c[i]], s[i]) で最大値を更新
- 詰まった点: `ans.at(c.at(i))` という二重の入れ子が出てこなかった。
  「添字が色番号を表す」という設計から必然的に出てくる形だった
- 学び:
  - カウント配列（バケット）の型: 値そのものを添字にして情報を溜める
    cnt[x]++ / sum[x] += v / mx[x] = max(mx[x], v) など応用が効く
  - 「今までで最大を覚える」型: ans = max(ans, new) は頻出
