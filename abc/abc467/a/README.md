# ABC467 A - BMI

- URL: https://atcoder.jp/contests/abc467/tasks/abc467_a
- 結果: AC（本番・自力）
- 解法: BMI = W / (H/100)^2 >= 25 を判定。小数と割り算を避けるため式を変形。
        W / (H/100)^2 >= 25
        → W >= 25 * (H/100)^2
        → W * 10000 >= 25 * H * H   （両辺 10000 倍して整数のまま比較）
- 学び:
  - H は cm。m に直すと H/100 だが、割り算・小数を避けて整数比較に変形するのが安全
  - BMI>=25 を W*10000 >= H*H*25 と両辺に 10000 を掛けて整数化
    （割り算の切り捨て・小数の誤差の両方を回避）
  - 「割り算で比較しそうになったら、両辺に掛けて整数比較にする」定番テク
