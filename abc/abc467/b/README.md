# ABC467 B - おつり

- URL: https://atcoder.jp/contests/abc467/tasks/abc467_b
- 結果: AC（本番・自力）
- 解法: 各店のお釣り = B - A。
        Y(全店で受け取った場合の損失総額) = 全店の (B-A) の合計
        X(実際に受け取った額) = take の店だけ (B-A) を合計
        答え Y - X = keep した店の (B-A) の合計（＝損した額）
- 反省: take を数えて最後に引くより、keep のときだけ (B-A) を足せば
        一発で損失が出た（if を keep で作る方が素直で速い）
    for ...: if (S == "keep") ans += B - A;
- 学び:
  - 「損した額」＝「お釣りを受け取らなかった(keep)店のお釣り合計」と直接読み替えられる
  - 求めたいものを直接数える方が、余計な変数(X,Y)を経由するより短くバグりにくい
