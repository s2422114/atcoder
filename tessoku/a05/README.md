# A05 - Three Cards（鉄則本 1章 計算量の改善）

- URL: https://atcoder.jp/contests/tessoku-book/tasks/tessoku_book_e
- 結果: AC（自力）
- 解法: x+y+z=K で z は自由に選べない。z = K-i-j で自動的に決まる。
        i,j の二重ループだけ回し、z が 1<=z<=N の範囲内なら1通りとして数える
- 学び:
  - 素直な三重ループは O(N^3)。N=3000 で 270億回 → TLE
  - 「1変数は他が決まれば自動的に決まる」→ z のループを消して O(N^2)（900万回）に改善
  - z = K-i-j が 1〜N の範囲に入るかを && でチェック（範囲外は数えない）
  - 数学的発想: 方程式 x+y+z=K の自由度は2。回すべき変数は2つで十分
  - 「三重ループになりそう→1つ減らせないか」は計算量削減の定石（鉄則6.5に対応）
