# A07 - Event Attendance（鉄則本 2章 いもす法）★3

- URL: https://atcoder.jp/contests/tessoku-book/tasks/tessoku_book_g
- 結果: AC（考え方は自力、配列サイズで1度RE→D+2で解消）
- 解法: いもす法。参加者ごとに B[L]++、B[R+1]-- で端だけマーク。
        最後に B を前から累積和 S[i]=S[i-1]+B[i] して各日の出席者数を復元
- 詰まった点と学び:
  - 【定石】いもす法は R+1 に -1 を置くので、配列を D+2 に大きく取る
    （最終日 R=D のとき B[D+1] にアクセス。D+1 サイズだと範囲外RE）
  - サンプルは通っても R=D のケースで落ちる（区間の端の見落とし）
  - 累積和(A06)=読み取りを速く / いもす法(A07)=区間加算を速く
  - いもす法は最後に累積和を使う（端に差分→累積和で展開の2段構え）
- 難易度: ★3、ABC C問題レベル。C問題頻出の武器
