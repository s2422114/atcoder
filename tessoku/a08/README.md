# A08 - Two Dimensional Sum（鉄則本 2章 二次元累積和）★4

- URL: https://atcoder.jp/contests/tessoku-book/tasks/tessoku_book_h
- 結果: AC（考え方は理解、実装ミスを複数修正）
- 解法: 二次元累積和。横方向→縦方向の順に累積和 Z を作る。
        長方形(A,B)-(C,D)の和 = Z[C][D] - Z[A-1][D] - Z[C][B-1] + Z[A-1][B-1]
- 学び:
  - 【包除原理】全体から上と左のはみ出しを引き、2回引いた左上の角を +1 で足し戻す
  - 累積和の作成も2段階: 横に Z[i][j]=Z[i][j-1]+X[i][j]、
    次に縦に Z[i][j]+=Z[i-1][j]
  - 1-indexed が必須級に楽（A-1, B-1 が 0 になっても Z[0][*]=0 で安全）
  - 【罠】vector<int> A, B, C, D(Q+1) は D だけにサイズが付く！
    → A(Q+1), B(Q+1), C(Q+1), D(Q+1) と個別に付ける
  - 【罠】縦累積和の内側ループは i<=H（W と取り違えやすい）
- 難易度: ★4。ABC C問題〜やや上。発想でなく実装の丁寧さが問われる
