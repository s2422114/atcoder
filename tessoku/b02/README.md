# B02 - Divisor Check（鉄則本 1章）

- URL: https://atcoder.jp/contests/tessoku-book/tasks/tessoku_book_ca
- 結果: AC（自力）
- 解法: A〜B を線形探索し、100 % i == 0（iが100の約数）が1つでもあれば Yes
- 学び:
  - 「Xの約数か」= 「X % 候補 == 0」（割り切れる＝余り0）
  - A02の線形探索の応用。探す条件が「一致」→「約数」に変わっただけ
  - 割り算/余りは割る数が0だとゼロ除算で落ちる。制約 1<=A が i>=1 を保証
    → 割り算時は「割る数が0にならないか」を制約で確認する癖を
