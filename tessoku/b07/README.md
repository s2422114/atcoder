# B07 - Convenience Store 2（鉄則本 2章 いもす法）

- URL: https://atcoder.jp/contests/tessoku-book/tasks/tessoku_book_cf
- 結果: AC（i=0の場合分けと -1 の位置を相談）
- 解法: いもす法。時刻は0始まり。B[L]++, B[R]-- （R退勤=R時にはいない）。
        累積和 S を取り、t=0〜T-1 の各時間帯(t+0.5)の人数を出力
- 学び:
  - 【重要】-1 の位置は「Rを含むか」で変わる:
      A07「R日目まで出席」= 閉区間[L,R] → B[R+1]--
      B07「R時に退勤、t+0.5で判定」= 半開区間[L,R) → B[R]--
    問題文の「まで/退勤/未満」を読んで、右端を含むか判断する
  - 0-indexed だと累積和の i=0 は前がない → S[0]=B[0] と別扱い（i=1から回す）
    1-indexed なら S[0]=0 で場合分け不要。時刻が0始まりなら0-indexedが自然
  - 配列の始まりは問題の番号の始まりに合わせる（人=1始まり、時刻=0始まり）
