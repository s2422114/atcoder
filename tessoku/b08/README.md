# B08 - Counting Points（鉄則本 2章 二次元累積和）★4

- URL: https://atcoder.jp/contests/tessoku-book/tasks/tessoku_book_cg
- 結果: AC（考え方を図解で理解、実装ミスを修正）
- 解法: 点をバケットで盤面に置く num[X][Y]++ → A08と同じ二次元累積和 →
        質問は S[c][d]-S[a-1][d]-S[c][b-1]+S[a-1][b-1]（包除原理）
- A08との違い: 盤面が最初からない。点の座標を num[X][Y]++ で自分で盤面化する
        （座標を添字にするバケットの二次元版）
- 詰まった点と学び:
  - 累積和ループの上限は配列サイズ内に（<=1500。1502だと範囲外）
  - 縦累積和は S[i][j]=S[i-1][j]+S[i][j]（S[i][i]はタイプミス→答え崩壊）
  - 座標最大1500 → 盤面も1500×1500必要（座標がそのままマス位置）
- 学習方針メモ:
  - 原理(包除で重なりを足し引き)は理解する→ 気づき・応用のため
  - 作り方と式は手に覚えさせる→ 速さのため
  - 自分用ライブラリ化して本番はコピペ（鉄則 終章「ライブラリを準備」）
