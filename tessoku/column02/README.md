# コラム2 - ビット全探索（鉄則本 1章コラム）

- 判定: AtCoder採点なし。書籍の解答例で自己確認
- 問題: N枚のカードからいくつか選び、合計を S(K) にできるか判定
- 解法: ビット全探索。0〜2^N-1 の各 i について、i の j 番目のビットが
        立っていればカード j を選ぶ、として合計を計算。S に一致すれば Yes
- 学び:
  - 各要素の「選ぶ/選ばない」を2進数の各ビットに対応 → 2^N 通りを全探索
  - N が変わるとforのネスト数が変わる問題を、bitループ1本で解決するのがビット全探索
  - ビット判定: (i / (1<<(j-1))) % 2 == 1（割り算と余りで桁を調べる）
    別解(ビット演算): (i >> (j-1)) & 1 で同じこと（>>で桁を下げ &1で最下位を取る）
  - 使える前提は N が小さいこと（2^N が現実的、目安 N<=20〜25）
  - A04の「%2と/で桁を見る」と同じ発想
